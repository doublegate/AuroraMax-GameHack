# Project Context

## Project Overview
AuroraMax-GameHack is a custom Linux distribution based on Universal Blue and Atomic Fedora. This project aims to create a specialized gaming-focused immutable desktop operating system.

## Key Information
- Based on Universal Blue (which itself is based on Fedora Atomic/Silverblue)
- Immutable/image-based operating system
- Gaming-focused customizations and optimizations
- Uses rpm-ostree for system management
